Develop Mobile App with Expo with Maps Feature

Node JS: v22.15.0 (LTS)

Initialize: npx create-expo-app@latest vino-mobile

NPM Dependency: 
npx expo install @react-navigation/native @react-navigation/stack react-native-screens react-native-safe-area-context react-native-maps expo-location

Run Project: npx expo start -c

Build APK: https://docs.expo.dev/build-reference/apk/
npm install --global expo-cli eas-cli



Refs:
https://docs.expo.dev/tutorial/create-your-first-app/
eas build -p android --profile preview
